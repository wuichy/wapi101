Empieza en 5 minutos

Crea una app, recibe credenciales OAuth, y haz tu primera llamada.

# 1) Tu cliente autoriza tu app (OAuth flow)
GET https://wapi101.com/oauth/authorize?client_id=tu_app&redirect_uri=...&scope=leads:read

# 2) Intercambias el code por un token
POST https://wapi101.com/oauth/token
  grant_type=authorization_code&code=...

# 3) Llamas la API con tu token
GET https://wapi101.com/api/expedients
  Authorization: Bearer wapi_at_xxx...
