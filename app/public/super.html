Email / SMTP

Configura cómo Wapi101 envía emails transaccionales (reset de contraseña, notificaciones de signup, etc.). Equivalente a WP Mail SMTP.

Solo aparecen las cuentas Gmail ya conectadas en el CRM (Integraciones → Gmail). Si no ves ninguna, ve al CRM y conecta primero tu cuenta Gmail.

Obtén tu key en resend.com/api-keys. Plan gratuito: 3,000 emails/mes.

Gmail: usa una contraseña de aplicación (2FA requerido). ZeptoMail: host smtp.zeptomail.com, puerto 587. Postmark: host smtp.postmarkapp.com, usuario = API Token.

Aquí llegan las notificaciones de nuevos signups y alertas del sistema.

El email de prueba se enviará a la dirección del admin configurada arriba.
